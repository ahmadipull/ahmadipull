Pengertian JavaScript

JavaScript adalah bahasa pemrograman populer yang digunakan untuk membuat situs dengan konten website yang dinamis. Faktanya, ada lebih dari 67,7 persen developer yang menggunakan JavaScript,lho! Konten dinamis artinya konten dapat bergerak atau berubah di depan layar tanpa perlu mereload halaman. Misalnya saja, fitur slideshow foto, gambar animasi, pengisian poling, dan lainnya.

Pengertian HTML

HTML adalah singkatan dari HyperText Markup Language. HTML adalah bahasa markup standar yang digunakan untuk membuat halaman web. Bahasa ini menggunakan tag atau elemen yang ditempatkan di dalam dokumen HTML untuk mendefinisikan struktur dan konten dari sebuah halaman web. HTML terdiri dari elemen-elemen yang membentuk struktur dasar halaman web. Setiap elemen memiliki fungsi dan makna tertentu

Pengertian CSS

CSS adalah singkatan dari Cascading Style Sheets. CSS adalah bahasa yang digunakan untuk mengatur tampilan dan penataan elemen-elemen dalam halaman web. Dengan CSS, Anda dapat mengubah tampilan visual, seperti warna, ukuran, tata letak, dan gaya teks dari elemen-elemen HTML CSS memisahkan tata letak dan penampilan dari struktur konten dalam dokumen HTML. Dengan menggunakan aturan CSS, Anda dapat memilih elemen tertentu dalam halaman web dan memberikan gaya atau properti tertentu kepada elemen tersebut. Aturan CSS terdiri dari selector (pemilih) yang menentukan elemen yang akan diberikan gaya, dan deklarasi yang berisi properti dan nilai yang akan diterapkan pada elemen tersebut.
